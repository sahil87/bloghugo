The Nodejs Toolbox (TNT)

December 20, 2016 Sahil Ahuja

1 minute read

  • Twitter

  • Facebook

  • Reddit

  • LinkedIn

  • Email

An opinionated list of great tools for NodeJS tooling.

Module bundlers
  • Browserify, works well with npm packages
  • RequireJS, works well with bower packages
  • jspm
Task Runners
  • Gulp 4.0

Gulp - The swiss army knife of task runners

Process Monitors
  • Nodemon
  • Supervisor
  • PM2
  • Forever
Watch mode live reload tools
  • Browsersync
  • livereload
  • watchify watch mode for browserify builds
Compilers
  • Babel
  • Webpack, webpack2 - also does all of the above.

Webpack - The swiss army knife of swiss army knives!
